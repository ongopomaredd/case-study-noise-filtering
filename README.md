#Case Study: Noise Filtering and Data Analysis for HALOBASE sensor

## Method
Median filtering spike removal preceded by Exponential Moving Average (EMA)

## Motivation
Cleaning the noise caused by the enviromental particle density changes, hopefully leading to better measurements that correspond to valuable feedback to improve sleep quality of the user. 

## Files
- casestudy.py - main filtering python script
- Case_Study_Report.pdf - report of the case study
- casestudyplot.png - main output plot comparing the phases of the process
- additional files - extra plots
